# Real Estate Filters (Vue 3)

Тестовое задание: простая страница поиска квартир в Астане с фильтрами.

## Функционал
- фильтрация по площади (от / до)
- фильтрация по количеству комнат (от / до)
- поиск по адресу
- адаптивный интерфейс

## Технологии
- Vue 3 (Composition API)
- Vite
- Bootstrap
- Локальный JSON

## Клонирование репозитория
```bash
git clone https://github.com/IslamK1/VueJS-TZ.git
cd VueJS-TZ
```

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте в браузере: http://localhost:5173
